Dynamically replaces localhost URLs with the local IP address.
== Description ==
In today's world, where collaboration and easy access to information resources are of paramount importance,
the Local Network WordPress Sharing Plugin offers an ideal solution for sharing and accessing WordPress
websites within local network environments such as educational settings, offices, and software development
environments. With its unique features, this plugin facilitates the process of sharing and accessing WordPress
websites on local networks in an unparalleled manner.
